Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplsih this analysis, we first pulled data from the OpenWeatheeMap API to assemble a dataset pn pver 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. Latitude. Factors we looked at included: temperature, cloudliness, wind speed and humidity. This site provides the source data and Visualizations created as part of the analysis, as well as explainations and descriptions of any trends and correlations witnessed.
